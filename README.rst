Django-hpcloud
==============


Django-hpcloud is a helper module intended for use with django to hook into the
HPCloud services.

You need to supply in your settings.py file:

* HP_SECRET_KEY
* HP_ACCESS_KEY
* HPCLOUD_USERNAME
* HPCLOUD_PASSWORD
* TENANT_ID
* OBJECT_STORE_URL
* REGION_URL
* CDN_URL

These can be found on your API Keys page.
